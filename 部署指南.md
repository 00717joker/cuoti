# 错题管理智能体 - 云端部署指南

## 目标
将网站部署到云端服务器，电脑不用开机，手机随时在外访问。

- ✅ 数据永久保存（SQLite）
- ✅ 无需配置域名，自动分配 `用户名.pythonanywhere.com`
- ✅ 无休眠，7x24小时在线
- ✅ 每月免费配额足够个人使用

## 部署步骤（5分钟搞定）

### 第一步：注册 PythonAnywhere
1. 打开 https://www.pythonanywhere.com
2. 点击右上角 `Start running Python online`
3. 注册账号（免费即可）

### 第二步：上传代码文件
1. 登录后点击顶部 `Files` 标签
2. 点击 `New directory` 创建文件夹：`mysite`
3. 进入 `/home/你的用户名/mysite/`
4. 点击 `Upload a file`，上传以下3个文件：
   - `app.py`
   - `requirements.txt`
   - `templates/index.html`

### 第三步：安装依赖
1. 点击顶部 `Consoles` 标签
2. 点击 `Bash` 打开终端
3. 输入命令：
   ```bash
   cd ~/mysite
   pip install flask
   ```

### 第四步：配置 WSGI
1. 点击顶部 `Web` 标签
2. 点击 `Add a new web app`
3. 选择 `Flask` → `Python 3.11`
4. 路径填：`/home/你的用户名/mysite`
5. 点击 `Next` → `Next` → `Finish`
6. 点击页面上的 WSGI 配置文件链接（如 `/var/www/用户名_pythonanywhere_com_wsgi.py`）
7. 修改为：
   ```python
   import sys
   sys.path.insert(0, '/home/你的用户名/mysite')
   from app import app as application
   ```
8. 点击 `Save`

### 第五步：启动网站
1. 返回 `Web` 页面
2. 点击顶部绿色 `Reload 用户名.pythonanywhere.com`
3. 等待 10 秒后，访问 `https://你的用户名.pythonanywhere.com`

## 常见问题

### Q: 上传文件后网站打不开？
A: 检查 WSGI 配置文件中的路径是否正确，确保 `/home/你的用户名/mysite` 正确。

### Q: 数据库数据丢失？
A: PythonAnywhere 的 SQLite 数据不会丢失，除非你手动删除文件。

### Q: 需要付费吗？
A: 免费版足够个人使用。如果流量不够，可以升级到付费版（约 ¥30/月）。
